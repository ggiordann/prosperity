# Tools needed

All algorithmic trading in Prosperity happens in Python. To get started with programming you will need to install a recent version of Python 3. You can find specific releases of Python on the Python downloads pageahttps://www.python.org/downloads/. Just scroll down a bit until you see a recent version of Python, e.g. Python 3.12, and install that.

> 

Next, you will need a way to write your Python script. It’s recommended to do this in a text editor or IDE that has syntax highlighting for Python. A popular IDE is VS Codeahttps://code.visualstudio.com/.

With Python installed, a good IDE and a nice cup of coffee or tea, you can start writing your program.
