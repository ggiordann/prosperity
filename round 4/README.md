# Round 4 Archive

This folder keeps the Round 4 work out of the active Round 5 workspace.

- `analysis/` contains reports, order-flow studies, visualization outputs, and engine artifacts.
- `current_champion/` contains the Round 4 champion submissions.
- `docs/` contains Round 4-specific notes.
- `scripts/` contains Round 4 search, visualization, and packing scripts.
- `src/` contains the archived `prosperity.round4_*` packages.
- `tmp_backtests/` contains generated Round 4 search variants and sweep outputs.
